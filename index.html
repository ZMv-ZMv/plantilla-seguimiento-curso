Plantilla de seguimiento de cusos

En este archivo se registrarán los cursos que se estén llevando o que ya se hayan llevado para diferenciarlos de otros, así como el avance que se hace de los mismos.

Este repositorio siempre permanecerá en la carpeta 'Practicando' ubicada en el escritorio, y dentro de este repositorio se registrará el avance de cada curso según convenga.

La estructura de las carpetas dentro de 'Practicando' y cómo se están llevando los cursos es:

  • Desktop
    • Pracitcando
      • _GNRL (donde se guardará este documento para el formateo general del seguimiento de cursos)
        • .git
        • css
        • img
        • js
        • index.html
        • readme.md
      • AutoCAD
      • Cpp
      • Excel
      • Python
      • WEB
      • ... otros temas (estos temas no son los cursos en sí, cada tema - carpeta tiene sus respectivos cursos dentro)

Dentro de la carpeta 'Practicando' se ubicarán carpetas por temas a practicar y dentro de estos temas los cursos propiamente dichos.

-TODOS- los cursos llevados deben tener un registro en estas carpetas dentro de su respectivo tema, sean cursos con origen en PC, Youtube, Udemy o le que sea, deben registrarse en esta carpeta para que quede como fuente de consulta a futuro y como verificación si un curso ya se llevo y no estar repitiendolo.

Al empezar un curso, dependiendo del origen del mismo se procederá de la siguiente manera:

  • 1. Con origen en PC: son cursos que han sido descargados o que los recursos se tienen almacenados de forma local, en la PC o en almacenamiento externo de algún tipo.

    • 1.1. En la ruta de los videos del curso (que no es la carpeta 'Practicando' sino en otra unidad de almacenamiento), se deberá incluir una carpeta llamada '_SEGIMIENTO-CURSO' en la que se deberá incluir un archivo 'nombre-de-curso.html'. Este archivo contendrá unicamente el 'ENCABEZADO DEL CURSO' (desarrollado más abajo), esto como indicativo de que el curso está siendo llevado.

    • 1.2. Luego, en la carpeta 'Practicando', dentro de sus respectivas carpetas de tema y nombre de curso, se incluirá este archivo 'nombre-de-curso.html' en el que además del encabezado del curso,se agregará todo el desarrollo del curso, incluyendo links a los videos, material de apoyo, archivos, documentación, apuntes, etc.

  • 2. Con origen Web (Youtube, Udemy, otros): de frente se incluirá el archivo 'nombre-de-curso.html' tal cual el punto 1.2. En este caso, si existieran videos, los link a éstos deberán dirigir a la web que pertenecen.
